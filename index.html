AcadePic

Prompt-centered academic figure generation.

AcadePic is a paper-to-figure studio for drafting prompts, generating candidate diagrams and plots, running manual critique, and iterating toward publication-ready academic visuals.

Open Prompt Studio GitHub Repository

Prompt Studio

Parse LaTeX sections, select content blocks, generate Planner and Stylist prompts, run candidate generation, and refine with Critic in a persistent studio workflow.

Diagram + Plot

Work in both diagram mode and plot mode. Plot mode includes a Matplotlib code workspace for viewing, editing, running, and downloading generated scripts and previews.

History-first workflow

Keep candidate generations, prompt revisions, and Critic results in persistent history so figure work can be resumed, compared, and archived over time.

How to run

  • Clone the repository and enter the project directory.
  • Run ./scripts/run_prompt_studio.sh.
  • Open http://127.0.0.1:8610/ or /studio.
  • Fill your own provider settings in configs/model_config.yaml or directly in the UI.
